# widgen-text-reminders

Get Started
```
clone https://github.com/williamrmyers/widgen-text-reminders
npm install
cd client
npm install
```

create file `config.json` in config folder.
```
{
    "development": {
      "PORT":8080,
      "MONGODB_URI": "",
      "JWT_SECRET": "",
      "TWILIO_ACCOUNT_SID": "",
      "TWILIO_AUTH_TOKEN": ""
    }
}
```



Start Dev Server
```
npm run dev
```
Test
```
npm test
```
